# HR Employee Attrition SQL Project

## Project Overview

This project analyzes employee attrition data using SQL to identify factors affecting employee turnover and workforce retention.

## Tools Used

* PostgreSQL
* pgAdmin 4
* SQL
* Excel

## Dataset

The dataset contains employee information such as:

* Age
* Department
* Job Role
* Salary
* Education
* Years at Company
* Attrition Status

## Key SQL Analysis

* Employee Attrition Analysis
* Department-wise Attrition
* Job Role Analysis
* Salary and Attrition Trends
* Employee Demographics Analysis
